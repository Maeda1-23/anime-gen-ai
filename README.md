# アニメ制作自動化システム

## 概要

このプロジェクトは、アニメ制作プロセスの一部を自動化するシステムです。Google Gemini APIを活用して、画像の理解と生成を効率的に行います。

## 特徴

- **VLM（Vision Language Model）による画像理解**: 画像の内容を分析し、説明を生成
- **画像生成AI**: テキストプロンプトから高品質な画像を生成
- **外部API活用**: ローカルモデルではなく、Google Gemini APIを使用することで、高い性能と拡張性を実現

## 技術スタック

- Python 3.x
- Google Gemini API
- Pillow（画像処理）

## セットアップ

### 前提条件

- Python 3.x
- Google Gemini APIキー

### インストール

```bash
pip install -r requirements.txt
```

### 環境変数設定

`.env`ファイルに以下を設定：

```
GEMINI_API_KEY=your_api_key_here
```

## 使い方

```bash
python main.py
```

詳細な使い方はドキュメントを参照してください。

## 開発の背景

このプロジェクトは、従来のローカルモデルによるアニメ制作自動化システムの問題点を解決するために開発されました。外部APIを使用することで、以下のメリットがあります：

- モデルの更新が自動的
- より高い性能のモデルにアクセス可能
- ローカルリソースの節約
